# Blackjack

Projekt je napisan u Jack jeziku.
Blackjack s ASCII prikazom karata u terminalu. 
Igrač može vući karte (hit) ili stati (stand), diler igra automatski po standardnim pravilima. Špil se miješa pseudo-nasumično.
Igrač počinje s 100$ i može staviti ulog od 5, 10 ili 25 dolara po rundi.

## Datoteke
- Card.jack / Card.vm - predstavljanje karte (boja i vrijednost)
- Deck.jack / Deck.vm - špil s pseudo-nasumičnim miješanjem
- Hand.jack / Hand.vm - logika ruke igrača i dilera
- Game.jack / Game.vm - glavna logika igre i prikaz
- Main.jack / Main.vm - ulazna točka programa

## Pokretanje
Potreban je Nand2Tetris VM Emulator. Učitaj sve .vm datoteke i pokreni.
